# 📋 Changelog — KuruAi

All notable changes to this project will be documented here.

## [4.0.0] — 2025-01-01 · *v4 Overhaul*

### ✨ New Features
- Complete UI redesign — Sora/DM Sans fonts, purple/pink color scheme, Flaticon icons
- OTP-based login & registration via Brevo email
- Subscription tier system: Free / Premium / VIP with daily usage limits
- Full Admin Dashboard (user management, ban system, activity log)
- Inbox / messaging system (admin → user broadcasts)
- Native Shop page with product card grid
- Nickname system — custom AI recognition per user
- Duitku payment gateway integration
- Vision support (image input in chat)
- Real-time image gallery for generated images
- Changelog management in admin panel

### 🔧 Fixes & Improvements
- Admin login OTP bypass fix
- Image generation URL handling fixed
- Shop page converted from iframe embed to native card grid (X-Frame-Options workaround)
- Download proxy endpoint for generated images
- Real-time gallery updates without page reload
